GMOJ J2469 【灌水】

发表于 2021-01-28

Description

一个宽度为 $n$ 的底部不平的水槽，位置 $i$ 的高度为 $h_i$，水槽的两端无限高。现在给出你 $q$ 个相互独立的询问，问你将位置 $x$ 的水灌至高度 $y$ 需要多少水。

Solution

这个菜鸡第一时间想到的是两个 $\log$ 的屑做法。

考虑二分出左边和右边的挡板位置，然后可以用线段树来检验二分的答案的合法性。然后预处理一下前缀和即可做到 $O(q \log^2 n)$。

将询问离线，运用单调队列可以做到 $O(q \log n)$。
