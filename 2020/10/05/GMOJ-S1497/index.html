GMOJ S1497 【景点中心】

Description

给你一棵有 $n$ 个点的树，点 $i$ 上有 $a_i$ 个人，每条边有一个边权 $c_i$，现在请你求出所有人到同一个点汇合时走的路程总长的最小值，并求出这个点的编号，如果有多个点满足要求，输出距离点 1 最近的那个点的编号即可。

Solution

换根 DP 题中一道比较简单的题目。

设 $f1_i$ 表示点 $i$ 的子树内的答案，$f2_i$ 表示点 $i$ 为根时 $f1_i$ 的值，然后显然可以根据 $f1_i$ 等几个东西来快速地求出 $f2_i$。

于是两遍 DFS 即可解决。
