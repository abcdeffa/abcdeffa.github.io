GMOJ S5020 【牛奶装瓶】

Description

给你 $n$ 个数，$m$ 个操作，支持区间加、区间除然后向下取整、区间最小值、区间和。

Solution

除了 「区间除然后向下取整」 这个操作比较难处理以外，其他操作都可以简单地用线段树来维护。

考虑如果当前递归到一个区间，区间内最大值和最小值减少的值一样，就可以把原操作变为区间减了。

这个的时间复杂度可证明为 $O(n \log n \log C)$，其中 $C = 10^9\text{（值域）} + 10^4 q$，但我并不会证明。
